Polycardia is a card game website based on the universe of various video games. It is developed using Python and Flask, with an SQLite database to store cards and users. The game allows users to collect cards, and in the future, build their own deck and participate in battles.

Technology
Python
Flask
SQLite
HTML
CSS
Features
Collect and manage cards in a database.
Create profiles and manage users.
Battle mode between decks (coming soon).
Timer to open a booster for free.
Currency to speed up the timer.
Quests to earn currency (coming soon).
