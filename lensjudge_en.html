An automatic judging system designed for programming competitions. It allows users to submit a code file in one of the following four programming languages: Python, Java, C, and C++. The system compares the output generated by the program with an expected response after compiling and executing the code. This project is developed in Java and aims to automate the evaluation of solutions proposed during competitions or programming exams. It provides an efficient solution for checking answers in various languages.

Technology
Java
Features
Supports 4 languages: Python, Java, C, and C++.
Automatic compilation and execution of submitted programs.
Compares outputs with an expected response.
Handles compilation and execution errors.
Detailed reports for each submission.
