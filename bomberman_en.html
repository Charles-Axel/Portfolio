An adaptation of the classic Bomberman game, developed in Java with JavaFX. The game involves placing bombs to destroy obstacles and eliminate enemies. The project includes player movement management, explosions, and collisions in a 2D environment. The JavaFX interface displays game elements dynamically and interactively, with simple controls for player movement and actions.
