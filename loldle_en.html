This project is a game inspired by "Wordle", but featuring characters from the League of Legends universe. The goal is to guess a League of Legends champion using clues that change with each turn. The project is developed in Java using JavaFX for the graphical interface and Derby JDBC for database management.

Technology
Java
JavaFX
Derby JDBC
Features
Interactive user interface with dynamic elements.
Database management for storing champions and scores.
Game logic with progressive clues for each attempt.
Colorblind mode that modifies certain colors.
