Inspired by the TV game show, this project allows users to guess the prices of items. It uses Python for logic and communicates with an API that provides the real prices of Amazon products. The frontend is created with HTML and CSS, offering an interactive interface where users can submit their guesses and get real-time feedback. We chose a Minecraft theme for sound effects and decorations, as we have the freedom to choose a fun style.
