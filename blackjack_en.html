This project implements the Blackjack card game against a computer. It uses Java and JavaFX to create a fluid interface where users can draw cards, stay, or request a new card. The Java backend handles the game logic, including score calculation rules and comparison with the computer's hand. The goal is to achieve a score of 21 or get as close as possible without going over this number.

JavaFX
Features
Interactive user interface with JavaFX.
Game logic management with automatic card drawing.
