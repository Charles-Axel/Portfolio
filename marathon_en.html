A travel journal management website. It allows users to create, edit, display, and delete travel entries using a CRUD system (Create, Read, Update, Delete). The site also includes interactive features, such as the ability to comment on and like other users' posts. This project is developed in PHP for backend management, with HTML and CSS for the user interface. It allows travelers to share their experiences and receive feedback from the community.

This project was done in a group of 7, in collaboration with students from another department (Design, Graphics, etc.), within a 36-hour time frame.

Technology
PHP
Laravel
SQLite
HTML
CSS
Features
Create, edit, and delete trips, stages, and add photos.
View the trip and each stage.
Filter trips by seas and oceans.
Comment and like other users' trips.
